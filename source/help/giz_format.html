'.GIZ' file format


S.F.T. Setup Gizmo generates files in a format similar to an INI file. The following entries apply:


[Page1] These entries provide information from 'Page 1' of the wizard. They include the following:

AppName The name of the application from the start page
CompanyName The company name from the start page
SourcePath Root path for source files for building the SETUP image
DestPath The destination path to which the image will be written
LicenseFile The fully-qualified path for the license text file
StartMessage The 'start message' string from the start page
AppPath Advanced setting - default destination path during SETUP
AppCommon Advanced setting - default 'common files' path during SETUP
StartMenuFolderName Advanced setting - default 'start menu' folder name for creating shortcuts
CertFiles Advanced setting - list of one or more 'Cert' file names to automatically install.
Upgrade Advanced setting - list of application names to 'upgrade' to the one you're building the installation for.

[FileList] These entries contain a list of files to be included in SETUP and their settings from Page 3 of the wizard. The order of the settings is as follows:

    FileName.ext=a,b,c,d,e,"f","g",h,I,j,k,l,m

Each of these parameters represents one of the settings from the various pages. They are:
Auto Delete flag'delete after exec'
Execute flag'exec after copy'
Overwrite flag'overwrite always'
Shared flag'system file'
Dest Pathif 'other' was specified
Group Namequoted string, for options
Other Path Namedestination path when 'other' was specified
OS Flagbit mask indicating which OS's it should be installed on
No Uninstallnon-zero if the file should NEVER be uninstalled
Register'Register Control' (SETUP calls 'DllRegisterServer()' [value = 1] or 'regasm.exe' [value = 2]). The value is assigned according to the setting of the 'Register Control' and 'Use .Net Reg' flags.
Add Shortcutnon-zero to create a shortcut for this file
Exec Argsarguments for 'exec after copy' files during setup
Uninst Argsarguments for 'exec after copy' files during uninstall

[AddShortcut] These entries contain a list of files to be given shortcuts during SETUP and their settings from Page 3b of the wizard. The format is:

    FileName.ext="Desc","File.LNK","StartPath","Args","IcoFile",IcoIndex,nCMDShow

Where "Desc" is the long description, "File.LNK" is the name of the shortcut, "StartPath" is either blank or the path in which to run the application, "Args" is either blank or the arguments to pass to the application, "IcoFile" is the name of the 'icon' file if the default icon is not to be used, 'IcoIndex' is the icon index within the file specified by "IcoFile", and nCMDShow is the 'show window' command sent to the application on startup (default is 1, 'show normal').

[GroupList] These entries contain a list of 'groups' associated with the install options. The format is as follows:

    Group Name="Short Description","Long Description"

The 'Group Name' string will appear in the 'FileList' entries that are members of the option 'group'. The 'short description' and 'long description' are displayed on the options dialog box during SETUP.

[RegistryList] These entries contain a list of registry entries to be created during SETUP, and removed during UNINSTALL. The format is a 3 digit sequential index followed by the registry key in '[]' , the value name (either '@' or a quoted string), and the registry string value (in quotes).
