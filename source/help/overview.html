Overview of S.F.T. Setup Gizmo


The 'S.F.T. Setup Gizmo' is a tool for programmers to very quickly generate a SETUP system for their applications. Although other SETUP systems are available, even at no cost, they take a great deal of time to learn how to use, even for the simplest applications.

That is the reason that the 'S.F.T. Setup Gizmo' was created: To make SETUP the easiest thing that you do. It is only a matter of placing all of the appropriate files into a single directory tree, then indicating how each of them is to be installed, and adding the appropriate registry keys. 'S.F.T. Setup Gizmo' then creates a set of install diskettes, and a 'SETUP.INF' information file, and places the programs 'SETUP.EXE' and 'UNINST.EXE' onto disk #1 (or the DISK1 directory of a multi-disk CD image). Optionally, you can include a license agreement as a standard text (.txt) or RTF (.rtf) file, which is then copied onto DISK #1 along with 'SETUP.INF' and displayed when the user runs SETUP (the user will have to agree to the license to proceed).

The compressed files directory on a CD ROM image will contain the SETUP.EXE, UNINST.EXE, SETUP.INF, and optional LICENSE.TXT files, in addition to the first cabinet (compressed) file. Cabinet files can be uncompressed using the 'expand' utility as found on every Windows setup CD since '95. The main reason that 'UNINST.EXE' has been left uncompressed is to make it very easy for a user to uninstall an application by inserting the 1st install CD without having to re-install the product FIRST just to get the UNINSTALL program. Many people inadvertently delete things, and may need to uninstall the software to repair their system. Since the 'INF' file contains an 'UNINSTALL' section, the 'UNINST.EXE' program will simply read the 'INF' file, delete the appropriate files, remove the appropriate sub-directories, and clean up the registry entries that you entered. As such, it is important to include ALL registry entries in the INF file.
